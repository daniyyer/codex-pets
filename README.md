# Codex Pets

Custom pets for the Codex desktop app.

## Naitang

`naitang/` contains a Codex desktop pet package generated from local cat reference photos.

To install it on another computer, copy the folder:

```text
naitang
```

into:

```text
C:\Users\<your-user>\.codex\pets\naitang
```

Then restart Codex desktop and select the pet from the pet menu.
